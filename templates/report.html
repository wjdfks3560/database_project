본 문서는 '현란밈장터' 프로젝트의 주요 기능과 구현 방식에 대해 설명합니다.

1. 사이트 사용 방법

회원가입 및 로그인

사이트 우측 상단의 '회원가입' 버튼을 통해 새로운 계정을 생성할 수 있습니다. 로그인 후에는 '찜', '주문내역' 등 개인화된 메뉴를 이용할 수 있습니다.

상품 검색

상단 검색창을 통해 원하는 상품을 쉽게 찾을 수 있습니다. 일반 검색어는 상품 제목과 지역을 기준으로, @ 기호로 시작하는 검색어는 판매자 닉네임을 기준으로 검색합니다.

상품 상세 정보 확인 및 찜하기

메인 페이지나 검색 결과에서 상품 카드를 클릭하면 상세 페이지로 이동합니다. 이곳에서 상품의 상세 설명, 판매자 정보 등을 확인할 수 있으며, '찜하기' 버튼을 통해 관심 상품을 저장할 수 있습니다.

상품 구매 및 주문 내역 확인

'구매하기' 버튼을 누르면 주문서 페이지로 이동합니다. 배송지 정보를 입력하고 결제를 완료하면 주문이 생성됩니다. 생성된 주문은 우측 상단의 '주문내역' 메뉴에서 언제든지 확인할 수 있습니다.

2. 주요 기능 구현 설명

기술 스택

  • 백엔드 (Backend): Python, Flask Framework
  • 데이터베이스 (Database): MySQL
  • 프론트엔드 (Frontend): HTML, CSS, JavaScript

핵심 로직

  • 데이터 관리: 사용자가 등록하는 상품, 찜 내역, 주문 정보 등 모든 데이터는 MySQL 데이터베이스에 저장됩니다. Flask는 mysql-connector-python 라이브러리를 통해 DB와 통신하며, 각 기능에 맞는 SQL 쿼리를 실행하여 데이터를 조회, 생성, 수정, 삭제(CRUD)합니다.
  • 찜 기능: wishlist 테이블을 사용하여 사용자와 상품 간의 관계를 저장합니다. 사용자가 '찜하기' 버튼을 누르면 서버는 해당 사용자의 userid와 상품의 product_id를 wishlist 테이블에 추가하고, '찜 취소' 시에는 삭제합니다.
